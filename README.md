# TestApp

TestApp - мобильное приложение для мониторинга новостей финансового рынка. 
В качестве API используется сервис [Finhub.io](https://finnhub.io).
Минимальная поддерживаемая версия iOS - 15.0.
Только для iPhone.
Имеет светлую и темную темы.

В проекте используется следующий стек:
1. Язык приложения - Swift
2. Написан полностью на SwiftUI
3. Архитектура MVVM
4. Использована Core Data
5. GCD, асинхронная загрузка изображений
6. TabView для организации перехода между разделами
7. NavigationView, Navigation Link для перехода между экранами
8. WebKit для отображения GitHub-репозитория
9. JSON в качестве источника данных для табличного представления

Приложение имеет 6 экранов, ниже преведены описания  скриншоты кождого из них.

## Новости

Экран новостей представлен с помощью ScrollView в виде списка кастомных вьюшек, нажатия на которые направляют на новый экран с детальной информацией.
